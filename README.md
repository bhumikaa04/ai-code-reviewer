# 🤖 AI Code Reviewer (OpenEnv RL Edition)

An automated code repair agent powered by **Reinforcement Learning (RL)** and **Large Language Models**.  
This project implements a custom Gym-like environment where an AI agent learns to iteratively fix broken Python code.

---

## 🚀 Overview

Traditional static analysis tools **identify** errors — but don't fix them.  
This project treats code repair as a **Markov Decision Process (MDP)**:

| Component     | Description |
|---------------|-------------|
| **State**     | The current (potentially broken) Python code |
| **Action**    | A suggested code fix generated by the LLM (Qwen2.5-Coder / Zephyr) |
| **Reward**    | Calculated based on syntax validity, test passing, and security improvements |
| **Goal**      | Reach an **Execution Score of 1.0** (all tests passed) |

---

## 🏗️ Architecture

The system consists of three core components:

1. **CodeEnvironment** (`env/code_env.py`)  
   Safely executes code, runs unit tests, and calculates rewards.

2. **Inference Agent** (`inference.py`)  
   LLM-driven controller that interprets error logs and suggests fixes.

3. **Gradio UI** (`app.py`)  
   User-friendly dashboard to visualize repair steps and rewards in real time.

---

## 📊 RL Reward Logic

Reward shaping is used to guide the agent effectively:

| Event                                        | Reward     |
|----------------------------------------------|------------|
| ✅ All unit tests passed                     | `+1.0`     |
| 📈 Improvement in complexity / security     | Positive delta |
| ❌ Syntax error or no improvement            | `-0.1`     |
| ⏱️ Step penalty (encourages faster fixes)   | `-0.02`    |

## 🛠️ Installation & Setup

To run this project locally or in your own Hugging Face Space:

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/ai-code-reviewer.git
   cd ai-code-reviewer
   ```

2. **Set up Secrets**  
   Add your `HF_TOKEN` in the Hugging Face Space Settings.

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Run inference**
   ```bash
   python inference.py
   ```
